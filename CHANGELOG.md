# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-02-26

### Added:
- **Main screen:** Wikipedia article for selected anniversaries and events
- **Main screen:** Scrolling

### Moved:
- **The query of "What day is it?"** to main screen

### Fixed:
- **Minor bug fixes**

### Removed:
- **Splash image on main screen**

## [1.1.0] - 2026-02-15

### Added:
- **Main screen:** App's logo to fill in the rest of the screen
- **Options:** Ability to set various date formats (common ones)
- **Drawer:** Show you the day name by date
- **Appbar:** Shows you current date, day number and remaining days of current year

### Fixed:
- **Calendar:** The way week starts are calculated
- **Various bug fixes**: For instance the way some dialogs are closed

### Changed:
- **Different app icon**
- **Licence changed to MIT**

## [1.0.0] - 2026-02-09

### Initial version
- **Core Engine:** Implementation of date-to-week-number logic using ISO 8601 standards.
- **Navigation:** Added ability to step forward and backward through weeks with automatic date range updating.
- **Manual Jump:** Users can now set a specific custom date or week number.
- **Calendar Flexibility:** Added support for choosing the week's start day (Sunday, Monday, or Saturday).
- **Theme Engine:** Implementation of Light and Dark mode support via Flet.
